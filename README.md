# bitcoin-news-impact-analysis
Автоматизированная система анализа новостных сообщений и оценки их влияния на курс Bitcoin с использованием методов машинного обучения и обработки естественного языка (NLP). Проект включает сбор данных, предобработку текста, определение тональности и анализ влияния новостей на динамику криптовалютного рынка.
