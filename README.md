# Анализ влияния новостей на курс Bitcoin

## Описание проекта

Данный проект представляет собой систему автоматизированного анализа новостных сообщений и оценки их влияния на курс Bitcoin (BTC) с применением методов машинного обучения и обработки естественного языка (NLP).

Система позволяет определять тональность новостей (позитивная, нейтральная, негативная) и исследовать её связь с изменениями цены криптовалюты.

## Цель проекта

Разработка программного модуля, обеспечивающего:

- сбор новостных данных из различных источников;
- предобработку и очистку текстовой информации;
- определение тональности новостей;
- оценку влияния новостных событий на курс Bitcoin;
- визуализацию результатов анализа.

## Архитектура системы

1. Сбор новостных данных (RSS, API, социальные сети)
2. Предобработка текста (очистка, токенизация, лемматизация)
3. Анализ тональности новостей
4. Формирование признаков
5. Оценка влияния новостей на цену Bitcoin
6. Визуализация результатов

## Выходные данные системы

- Классификация тональности новостей
- Уровень уверенности модели
- Оценка влияния новости на цену BTC
- Анализ временных задержек (5 мин, 30 мин, 1 час)
- Графики зависимости «новости — цена»
